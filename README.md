## Run locally
1. Unzip
2. In project root:
   ```
   npm install
   npm run dev
   ```
3. Open http://localhost:5173

## Customization
- Update phone/email in `src/i18n.js`
- Replace images in `src/App.jsx` IMAGES object
- Replace logo text / add real logo in Nav
